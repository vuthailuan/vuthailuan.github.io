# Vu Thai Luan GitHub Pages Website

This is a static GitHub Pages website rebuilt from the original Word-exported HTML source files.

## Publish

1. Create a GitHub repository named `username.github.io`.
2. Upload all files and folders from this zip to the root of the repository.
3. In GitHub, go to **Settings → Pages** and publish from the `main` branch, root folder.

## Notes

- The page content is rebuilt from the source files in the uploaded zip.
- The design uses a single horizontal navigation bar and removes duplicated page lists/sidebar links.
- The original uploaded source zip is saved at `assets/source/original-website-source.zip`.
